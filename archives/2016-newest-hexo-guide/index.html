Linux下安装node.js时

Windows上使用Hexo deploy提交到github上时报”Error: spawn ENOENT”

这是git环境变量设置的问题，解决方法就是不要在cmd上执行hexo deploy命令，而是安装git后在git bash的命令行中执行hexo deploy
http://zhenghongen.cn/2013/07/09/hexo-deploy-throw-error/

打开md文件，编辑文章内容

创建菜单项，实现url映射

分页样式：https://hexo.io/zh-cn/docs/helpers.html

安装git插件，使用ssh提交到github npm install hexo-deployer-git –save

配置：
deploy:
type: git
repo: ssh://github.com/user-name/user-name.github.io #换成自己的用户名
branch: master

windows下如何github ssh 公钥

  1. 从开始菜单中打开 “Git Bash”
  2. 在根目录输入命令：ssh-keygen -t rsa -C “email@email.com” 其中，”email@email.com”是你的github账号
  3. 提醒你输入key的名称，输入如id_rsa
  4. 在根目录下产生两个文件：id_rsa和id_rsa.pub
  5. 把4中生成的密钥文件复制到“根目录.ssh\” 目 录下。
  6. 复制id_rsa.pub文件的内容，在github.com的网站上到ssh密钥管理页面，添加5复制的新公钥
