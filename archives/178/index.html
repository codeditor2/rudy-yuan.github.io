模板类的成员初始化方法

C++中对数据成员的初始化工作一般在构造函数中用赋值语句进行。

但是，对于常量类型和引用类型的数据成员，又不能在构造函数中用赋值语句直接赋值 （为什么？？？）

为此，C++提供了成员初始化列表置初值的方式！

PS：注意！成员初始化列表并不是函数原型的一部分！！

而static成员的初值必须在类体外进行！！（原因在于static成员可以在对象定义前就被访问，是在编译时创建和初始化）

结果为：

5.30更新

类的static数据成员赋初值时，需要指出该数据成员的类型，但是前面不能再加static。

比如，类Ctype中的某个static成员：static int s，那么，初始化时的语句为：int Ctype::s = 0;

当static成员为私有时，可通过公有static成员函数进行访问！！！
